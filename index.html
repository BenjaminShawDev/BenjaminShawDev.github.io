    Pacman Game

    A simple game with the theme of Pacman but plays slightly differently

    The goal is to collect all 50 randomly spread out munchies on the screen however there are no walls, the ghosts will travel across random parts of the screen and there are powerups that can be picked up

    You can also use your mouse to move a box which deflect ghosts to help protect Pacman from being hit

    Github link

    Mario Game

    A short and simple Mario game made in SDL consisting of two levels and two players controlled via a single keyboard

    To win the game you have to collect all of the coins on the second level without losing both Mario and Luigi

    Github link

    Asteroid Game

    A 3D game made in OpenGL where you get to control a space ship in an asteroid field

    You have to dodge oncoming asteroids and survive as long as possible, there are also different powerups that your ship can pick up in order to help you

    Github link

    Knight Game

    A simple SDL game I made over the 2020 summer which has no objectives but was used to experiment with SDL

    There is a grass and rock level which are both generated via a png file and the water tiles have collision

    The camera also follows the player and both the player and the camera stop at the edges of the map

    It was made with a mix of what I had learnt and an online tutorial which helped with aspects like an entity component system
