Networking Application

  • GitHub

  • Project GitHub page

A chat application that sends packets to and from a server to multiple clients and utilises multithreading to handle multiple clients and windows simultaneously. Using Windows Forms I created a basic GUI which easily allows the user to send packets containing messages to all the other clients or a specific client.

The server and each of the clients run on their own threads to reduce concurrency issues like freezing until a server response is received and it helps improve performance. There is a paint window which allows the clients to paint on the same canvas by sending packets with which pixels coordinates had been drawn on and with what RGBA values to use.

Possible improvements that could be made is moving the packet handling to the client instead of the server and adding the ability to send off more types of packets such as ones for encrypted messages.
