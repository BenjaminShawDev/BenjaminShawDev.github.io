DirectX Project

  • GitHub

  • Project GitHub page

A framework created with DirectX 11 in which the user can create and transform different textured 3D objects. The user can move around a 3D space and look around with basic mouse input with the option to switch between different cameras, which are each stored in an array, for different perspectives.

Pressing the left mouse button creates textured objects in front of them which are loaded from OBJ and DDS files. The object that is loaded can be changed and so can its material. Pixel shading is implemented with the use of a HLSL file which is affected by the object material. The colour of the light can be changed as well.

Loaded objects can be transformed and removed in the order that they were created with the ability to undo the most recent deletion.
