2nd Year Group Project - Molecularity

  • GitHub

  • Project GitHub page

A short puzzle game where the player manipulates a cubes properties to complete the level. The properties you can modify are the material, size, bounciness and conductivity. Each resulting in the cube behaving differently. It has basic physics and collisions for the cubes, with properties like mass being changed by its material and size which also affects its other properties like how well it can bounce.

The game has both 2D and 3D sounds setup using a singleton class and only loads the sounds it needs for each level using the level system which utilises multithreading. There is also a fully functioning UI system for the main menu, pause menu, the settings and the HUD. The settings make use of JSON files to read and save all the settings. We also had an event system to handle many of the events throughtout the game and to keep the code more organised while not linking multiple systems together.

Personally I worked on many differnet areas but mainly focused on getting sounds and physics working working and optimised.
