OpenGL Asteroids Project

  • GitHub

  • Project GitHub page

A 3D game made in OpenGL where you get to control a space ship in an asteroid field. This was my first 3D project and it really helped me understand the basics of OpenGL and creating a game in a 3D space.

There are a set amount of asteroids in the game and when they go behind the camera they are translated back to the front of the game and with the use a random number generator are given random X and Y coordinates. ollision detection was done by taking the position of the player and each asteroid and using 3D Pythagoras' thereom to calculate the distance between them and if it was smaller than the radius of the ship it would class as a collision.

In the future if I try something like this again I might try adding in better UI instead of a menu image and text with a 3D position which caused lighting to effect it. Also to also split the code into more classes as it mostly in the main class.
