Placement Project

A fully working game made entirely by me using DirectX 11 during my placement year. It is a rougelike style game with the aim being to survive as many rooms as possible. There are 3 playable classes and over 100 items.

Made an item and card system where items could be added to the player to make them stronger and cards could be added to make the game harder in different ways. There are different types of enemies throughout the game which use the A* algorithm for movement and there is a boss at the end of each floor with their own unique mechanics.

The game uses XAudio 2 for both 2D and 3D sounds in a singleton class. Level layouts and item information is loaded in from .xml files. Created the lighting and a UI system using HLSL which is used for the main menu, options menu and the HUD. Also made a text system that automatically aligns itself in the UI.
